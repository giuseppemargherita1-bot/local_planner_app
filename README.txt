MANPOWER PLANNER - versione desktop portabile

Avvio rapido:
1) apri ManpowerPlanner.exe
2) si apre il planner su http://127.0.0.1:8765

Struttura:
- Planner: vista principale stile fabbisogno, per commessa/mansione/settimana
- Risorse: anagrafica completa delle persone
- Gantt: carico risorse per periodo
- Report: riepiloghi settimana e settimana+mansione

Uso consigliato:
- primo avvio: se vuoi partire dai file Excel, usa il pulsante "Import iniziale"
- dopo l'import o anche senza import, lavora direttamente dentro il software
- nel planner clicca una cella oppure trascina una risorsa sulla week desiderata
- per assegnare piu settimane usa "Week da" e "Week a"

File principali:
- ManpowerPlanner.exe = launcher portabile
- planner.db = database locale
- app.py + static = logica applicativa
- pyembed = runtime Python incluso
